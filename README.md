Aplicação web que sorteia aleatoriamente personagens do universo de Harry Potter e exibe suas informações na tela.

O projeto consome uma API pública para obter os dados dos personagens e utiliza JavaScript para selecionar e mostrar um personagem aleatório de forma dinâmica.

Um dos principais focos do desenvolvimento foi garantir que a interface fosse **responsiva**, funcionando corretamente em diferentes tamanhos de tela, como celulares, tablets e desktops.

# ✨ Funcionalidades

- 🎲 Sorteio aleatório de personagens
- 🖼️ Exibição da imagem do personagem
- 🏰 Informações como casa e ator
- 📱 Interface totalmente responsiva
- ⚡ Atualização dinâmica ao clicar no botão
- 🌐 Consumo de API externa

---

# 📱 Responsividade

A interface foi construída com foco em **responsividade**, garantindo boa experiência em diferentes dispositivos.

Foram utilizados:

- **Flexbox** para organização e centralização do layout
- **Media Queries** para adaptação em telas menores
- **Imagens flexíveis**
- **Container responsivo**

O objetivo foi criar uma experiência funcional tanto em **smartphones quanto em desktops**.

---

# 📡 API utilizada

Este projeto consome dados da seguinte API:

https://hp-api.onrender.com/api/characters

A API fornece informações como:

- Nome
- Casa
- Ator
- Imagem
